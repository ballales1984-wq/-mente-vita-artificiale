================================================================================
TROUBLESHOOTING
================================================================================

Problema: "git: command not found"
Soluzione: Installa Git da https://git-scm.com/download/win

Problema: "Permission denied (publickey)"
Soluzione: 
  1. Genera SSH key: ssh-keygen -t ed25519 -C "your-email@example.com"
  2. Aggiungi su GitHub Settings -> SSH Keys
  
Problema: "Repository not found"
Soluzione: Verifica URL remote con: git remote -v

Problema: "Failed to push"
Soluzione: git pull --rebase origin main, poi git push

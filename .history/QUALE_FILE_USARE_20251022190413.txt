================================================================================
                    QUALE FILE USARE? - Guida Rapida
================================================================================

Hai MOLTI programmi nel progetto. Quale scegliere?

================================================================================
IL PIU' SEMPLICE (10 secondi)
================================================================================

> python esempio_semplice.py

COSA FA: Demo velocissima, 1 ciclo cognitivo
QUANDO USARLO: Prima volta, vuoi vedere subito come funziona
HARDWARE: No
DURATA: 10 secondi

================================================================================
IL PIU' COMPLETO (consigliato) ⭐⭐⭐
================================================================================

> python mente_artificiale_completa.py

COSA FA: TUTTO! Sistema completo con menu interattivo
INCLUDE:
  ✅ Tutti i 12 moduli cerebrali
  ✅ Camera + Microfono (se disponibili)
  ✅ Memoria intelligente
  ✅ Biosegnali neurali
  ✅ Apprendimento online
  ✅ Menu per scegliere modalità
  ✅ Espansioni pronte (voce, movimento, sensori, cloud)

QUANDO USARLO: Sempre! È il programma definitivo
HARDWARE: Opzionale (funziona anche senza)
MENU: Scegli 1-6 cosa fare

================================================================================
PER VISUALIZZARE (dashboard web) 📊
================================================================================

> streamlit run dashboard.py

COSA FA: Dashboard web con grafici e timeline
INCLUDE:
  - Percezioni real-time
  - Memorie
  - Emozioni
  - Grafici
  - Pulsante demo integrato

QUANDO USARLO: Vuoi vedere visualmente cosa succede
BROWSER: Si apre automaticamente su http://localhost:8501

================================================================================
PER TEST SENZA HARDWARE (demo simulata)
================================================================================

> python mente_ai_demo.py

COSA FA: Genera episodi simulati
QUANDO USARLO: Test dashboard senza camera/microfono
DURATA: Continuo (CTRL+C per fermare)

COMBINATO CON DASHBOARD:
  Terminal 1: python mente_ai_demo.py
  Terminal 2: streamlit run dashboard.py
